# cicd-intern-lab

CI/CD Basics with GitHub Actions — Intern Hands-On Lab

A Node.js Express API with automated testing (Jest) and continuous deployment to Render.com via GitHub Actions.

## Endpoints

- `GET /` — Welcome message
- `GET /health` — Health check
- `GET /add/:a/:b` — Add two numbers
- `GET /goodbye` — Goodbye message

## Run Locally

```bash
npm install
npm start
```

## Run Tests

```bash
npm test
```
